Page 2 of this example includes a script with `data-turbolinks-track="reload" in the head.
Turbolinks sees this while merging the head, and triggers a regular page load.
This leads to two requests sent to the server, via XHR and window.location

    OBS:

  • XHR and window.location requests are sent, hitting the server twice.
  • Page reloading is also triggered when navigating away from the page.
  • "TrackReload Script!" log only fires the first time page 3 is loaded.
  • Pages 4 & 5 have identical scripts in the head, and are NOT reloaded.
